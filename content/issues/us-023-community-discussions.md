## User Story

As a Dancer,
I want to participate in discussions,
So that I can share knowledge, ask questions, and learn from other community members.

## Acceptance Criteria

- [ ] Discussion features:
  - Create topics
  - Reply threads
  - Rich text formatting
  - Media embedding
  - Topic categories
- [ ] Organization tools:
  - Topic tags
  - Search/filter
  - Sort options
  - Bookmarks
  - Follow topics
- [ ] Interaction features:
  - Upvote/downvote
  - Best answers
  - Pinned posts
  - Closed topics
  - Topic subscriptions
- [ ] Moderation tools:
  - Content guidelines
  - Report system
  - User reputation
  - Moderation queue
  - Auto-moderation
- [ ] Community features:
  - Expert flairs
  - Achievement badges
  - Activity stats
  - Topic suggestions
  - Featured discussions
